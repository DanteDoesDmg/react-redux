# Redux learning repository

link: https://dantedoesdmg.github.io/react-redux/

## Functionality

1. App fetches posts from https:// jsonplaceholder.typicode.com/ and displays them
2. Writing a post and clickig 'Add post' will make a POST request, and display the new post on top of the list.

## Technologies used:

1. React
2. Typescript
3. Redux
4. ES6
